# Mpi (Materias primas e insumos).
El paquete Mpi (Materias primas e insumos) de V12 es el módulo que permite la administración del catálogo de materiales, insumos y materias primas, además de la gestión de líneas específicas para materias primas. Su propósito es garantizar un control eficiente de los recursos básicos de producción, asegurando disponibilidad, trazabilidad y coherencia con los demás procesos del ERP.
